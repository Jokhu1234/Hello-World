Below is a Definition list

Show Command Palette

Ctrl+P
Quick Open, Go to File

Ctrl+C
Copy line (empty selection)

Ctrl+Shift+[
Fold (collapse) region

Ctrl+Shift+]
Unfold (uncollapse) region

Ctrl+/
Toggle line comment
