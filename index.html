Visual Studio Code Extensions:

Below is an Ordered list

  • Prettier
    • is an opinionated code formatter

  • Live Server
    • makes your existing server live

  • vscode-icons
    • is a visual studio code extension that provides a set of icons

  • Auto Close Tag
    • a self-closing tag is a tag that does not need to be closed manually

  • Auto Complete Tag
    • autocomplete allows the browser to predict the value

  • Auto Rename Tag
    • every time you rename a tag, the complimentary opening/closing tag is renamed by itself to match the new tag name

    Visual Studio Code Keyboard Shortcuts Explanation

    Below is a Definition list

    Ctrl+Shift
    Show Command Palette

    Ctrl+P
    Quick Open, Go to File

    Ctrl+C
    Copy line (empty selection)

    Ctrl+Shift+[
    Fold (collapse) region

    Ctrl+Shift+]
    Unfold (uncollapse) region

    Ctrl+/
    Toggle line comment
